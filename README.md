# Youtube downloader
## PWA to allow you to download either mp3 or mp4 from YouTube!
### How to download?
1. Simply press on *CODE* and select *download ZIP*. 
2. Then unzip the folder and open it. 
3. Find the folder named *dist*.
4. Open it and there double-click on app.exe. 
5. Hold *CTRL* and press the 1st address (Running on http://127.0.0.1:8080).
